Divis√£o de treinamento e teste

Como o objetivo de comparar modelos para uma melhor previs√£o do o3, vamos dividir a base de dados em treinamento e teste.

data_o3_train<-
  data_o3 %>%
    mutate(week = wday(data,label = T)# %>% as.factor()

data_future<- expand.grid(data = max(data_o3$data)+1:90) %>% as.data.frame() %>%

# train the rf model

# train the ml model
set.seed(7)
modelo_ml <- train(Qtd_o3 ~ .,data=data_o3_train %>% select(-data) ,method = "mlpWeightDecayML",tuneLength = N)
